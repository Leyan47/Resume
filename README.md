### 🧰 技術內容

- 純前端製作（HTML + CSS）
- 響應式設計，支援暗黑模式
- 內嵌 PDF 瀏覽（使用 `<iframe>`）
- 無框架、無 JS，適合簡單展示與快速載入

---

### 🚀 部署方式

本網站已透過 GitHub Pages 部署，若你想 fork 並自行修改，可參考以下步驟：

1. Fork 本 repo
2. 將 `resume.pdf` 放到 repo 根目錄
3. 編輯 `index.html` 內容
4. 在 GitHub → Settings → Pages，設定 `main branch` 為來源
5. 等候幾分鐘，即可透過 `https://yourname.github.io/yourrepo` 存取
